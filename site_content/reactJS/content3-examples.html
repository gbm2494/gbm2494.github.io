Example Mockup

Application mockup: app that displays a list of news obtained from a JSON object.

The application should display a list of news from a JSON object, this list has to be displayed as a sidebar at the left. When a user clicks on a list item the information from this new has to be shown in the right side of the page.

Application should be divided on reusable components.

The application has to be divided in different components (ReactJS produces individual components). Components can be:

  • Component that displays the clicked item.
  • Single item for a new on the sidebar.
  • News list conformed by multiple single items.
