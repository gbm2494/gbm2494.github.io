General Objective

Develop skills needed on the student to implement a basic well structured application using ReactJS framework with Redux.

Specific Objectives

  1. Define Model-View-Controller pattern and how this pattern is applied in an application created with ReactJS and Redux.
  2. Identify general functionality from ReactJS in order to create graphical user interfaces.
  3. Identify basic characteristics from Redux as an application state container.
  4. Build a basic application that solves a particular situation which allows to see advantages from ReactJS and Redux over other JavaScript frameworks.

Content and Methodology

The course is divided in two stages, in the first one the student is going to learn about the basics of ReactJS and how the Model-View-Controller pattern works with it. Also, the students will learn about how to implement projects in ReactJS with pure javascript and with JSX language.

Model-View-Controller:
  • Definition of MVC:
    • Model
    • View
    • Controller
  • MVC on ReactJS
JSX language basics:
  • What is JSX?
  • Pure JavaScript vs JSX language
  • JSX basics
Introduction to ReactJS:
  • What is ReactJS?
  • Implementation of ReactJS with pure Javascript vs JSX language
  • Bundle.js file
  • How does ReactJS manipulate DOM?
  • How to start using ReactJS?
  • Functional components
  • Class based components
    • Component state
    • Constructor function
    • Render function

The second stage of the course is about Redux and how it helps ReactJS to create a complete MVC application. The student will learn how to implement a complete app with these technologies and will learn about some libraries that are useful for this kind of projects.

Model-View-Controller:
  • Redux layers:
    • Actions
    • Reducers
    • Store
  • ReactJS-Redux communication
Introduction to Redux:
  • What is Redux?
  • How to start using Redux?
  • From component to container
  • Designing actions and action creators
  • Middlewares inside the store
  • Asynchronous calls with Axios library
  • Creating reducers
Advanced React and Redux topics:
  • React Router
  • Redux Form

Project

The student is going to develop an application using ReactJS and Redux where he needs to use all knowledge he obtained during the course. He is going to create a kind of "Tinder" application where the user can choose what animals he prefer from a list of options the application is going to display. Also the user can manage his choices and administrate what kind of animals he wants to see according to his preferences. The teacher is going to explain all requirements and mockups to guide the development process.
